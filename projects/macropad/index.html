Project Description

This custom 20-key macro pad is powered by the RP2040 microcontroller and designed for maximum versatility. Each key is fully programmable, allowing for custom shortcuts, macros, or media controls across Windows, macOS, and Linux. The pad features a custom PCB, USB connectivity, and optional per-key RGB lighting, making it ideal for productivity, creative workflows, or gaming.

Project Details

  • Programmable keys allow shortcuts, macros, or controls to be customized.
  • Hotswappable MX switches allows you to customize your typing feel.
  • RGB Backlighting allows per-key lighting using WS2812/NeoPixel LEDs
  • MacroPad runs on a custom RP2040 PCB designed in Kicad.
  • Custom 3d printed case provides a sleek, ergonomic look.
