Project Description

This is a custom flight controller PCB I designed and built based on the STM32F1 chip. The PCB houses a BNO055 absolute orientation sensor to allow the flight controller to know its orientation in space, and breakout connections for the ESC, a downwards lidar for altitude control, a gimbal for camera/sensor control, a small OLED screen, and any other I2C connection.

Project Details

  • Based on an STM32F1 Chip with a BNO055 absolute orientation sensor for steady flight.
  • Easily connects to ESCs, lidars, gimbals, and more.
  • Designed to stack with standard 30.5 x 30.5mm stackup.
  • Easily programmed via SWD connection.
