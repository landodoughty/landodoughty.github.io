Project Description

This responsive portfolio website was built to showcase my engineering projects, technical experience, and contact information in a clean and accessible format. Using HTML, CSS (Bootstrap), and GitHub Pages, the site is fully custom-coded and organized to highlight both visual and functional aspects of each project.

Project Details

  • Custom-built using HTML, CSS, and Bootstrap for a clean, responsive layout.
  • Hosted on GitHub Pages for easy access and version control.
  • Project pages with image carousels for easy project showcases.
  • Mobile-friendly design with navigation optimized for all screen sizes.
  • Expandable structure for future projects and updates.
